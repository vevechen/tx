DCF估值结论

基于DCF模型，泡泡玛特每股内在价值约为234港元（基准情景），对应2026年经调整净利润约45倍PE。当前股价若低于200港元，具备一定安全边际；若高于280港元，则估值偏高，需更高的增长预期支撑。

核心假设风险：终值占比高达60%，对永续增长假设敏感；短期增速假设基于管理层指引，若指引下调估值将相应下调；WACC假设中Beta和ERP存在主观判断。

八、DCF模型方法论
# 自由现金流 (FCF) 计算公式
FCF = EBIT × (1 - 税率) + 折旧摊销 - 资本支出 - 营运资金增加
FCF = NOPAT + D&A - CAPEX - ΔWC

# 企业价值 (EV) 计算公式
EV = Σ[FCFt / (1 + WACC)t] + TV / (1 + WACC)n

# 终值 (Terminal Value) 计算公式 - Gordon增长模型
TV = FCFn × (1 + g) / (WACC - g)

# 股权价值计算公式
Equity Value = EV + 现金及等价物 - 有息负债 - 少数股东权益

# WACC计算公式
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1 - 税率)
Ke = Rf + Beta × ERP

模型优点

  • 基于现金流而非会计利润，更反映真实价值创造
  • 可灵活调整增长假设，适应不同情景分析
  • 终值占比60%符合高成长公司特征
  • 轻资产模式使FCF与净利润接近，模型可靠性高

模型局限

  • 对长期增长假设高度敏感（终值占比高）
  • WACC参数（Beta、ERP）存在主观判断
  • 未考虑实物期权价值（新业务拓展可能性）
  • 假设永续增长可能高估长期价值
